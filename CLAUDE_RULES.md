# RÈGLES KATY COUTURE - À RESPECTER TOUJOURS

## RÈGLE 1 : AFFICHAGE IDENTIQUE SUR TOUS LES ÉCRANS

Le site doit être IDENTIQUE peu importe :
- La taille de l'écran
- Le ratio de l'écran (16:9, 21:9, 4:3, etc.)
- L'appareil (téléphone, tablette, laptop, PC, ultrawide, etc.)
- La résolution

### Ce qui est autorisé :
- Agrandir/réduire les tailles proportionnellement
- Adapter les espacements (padding, margin)
- Adapter la largeur des éléments à l'écran

### Ce qui est INTERDIT :
- Changer la disposition selon l'écran
- Changer l'alignement selon l'écran
- Faire X colonnes sur un écran et Y colonnes sur un autre
- Cacher des éléments sur certains écrans
- Changer l'ordre des éléments
- Tout @media query qui change le layout

### Principe :
TOUS les utilisateurs voient EXACTEMENT la même chose, peu importe leur écran.
Seule la TAILLE change, jamais la DISPOSITION.
